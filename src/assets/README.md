# Assets Folder

## Image Required

Please place the `lady-justice.jpg` image file in this folder.

The image should be a high-quality JPG file showing Lady Justice (the symbol of law and fairness) - typically a statue or illustration of a woman holding scales and a sword, often blindfolded.

**Current Status:** A placeholder SVG file (`lady-justice.svg`) is included as a fallback. Once you add `lady-justice.jpg`, it will be used instead.
